🛡️ AdGuard LATAM

Política de Privacidad

Resumen: AdGuard LATAM no recopila, almacena ni transmite ningún dato personal. Todo el análisis ocurre localmente en tu navegador.

1. Qué es AdGuard LATAM

AdGuard LATAM es una extensión para Chrome que analiza la configuración de campañas en Meta Ads Manager y muestra advertencias sobre posibles errores antes de activar una pauta publicitaria. Opera exclusivamente dentro del navegador del usuario.

2. Datos que NO recopilamos

AdGuard LATAM no recopila, no almacena y no transmite:

  • Nombre, correo electrónico ni ningún dato de identificación personal
  • Datos de campañas, presupuestos, audiencias ni configuraciones de Meta Ads
  • Historial de navegación
  • Cookies ni identificadores de seguimiento
  • Información de cuentas de Facebook o Meta

3. Almacenamiento local

Los resultados del escaneo se guardan temporalmente en chrome.storage.session — un almacenamiento local del navegador que se elimina automáticamente al cerrar la pestaña. Ningún dato sale de tu dispositivo.

La configuración de reglas de validación se guarda en chrome.storage.local para evitar descargas repetidas. Esta configuración no contiene datos del usuario.

4. Configuración remota

La extensión descarga periódicamente un archivo de configuración desde GitHub para mantener actualizadas las reglas de validación (presupuestos mínimos, objetivos, eventos de conversión). Esta descarga:

  • No envía ningún dato del usuario a GitHub ni a ningún servidor
  • Solo descarga reglas genéricas de validación (no personalizadas por usuario)
  • Ocurre máximo una vez cada 6 horas

5. Telemetría de errores técnicos

La extensión incluye un sistema de telemetría para detectar cuando Meta Ads Manager actualiza su interfaz y rompe los detectores. Si se activa, este sistema envía únicamente:

  • Versión de la extensión instalada
  • Lista de detectores que fallaron (ej. "moneda", "presupuesto")
  • Hash criptográfico (SHA-256) de la ruta URL — no la URL completa
  • Marca de tiempo

Actualmente desactivado   Este sistema está desactivado por defecto. Si se activa en una versión futura, se notificará en las notas de actualización.

6. Permisos del navegador

La extensión solicita los siguientes permisos y los usa exclusivamente para:

  • Acceso a adsmanager.facebook.com: leer la configuración visible de campañas para mostrar advertencias. No modifica ningún dato.
  • storage: guardar resultados del escaneo y configuración de reglas localmente.
  • alarms: programar la descarga periódica de la configuración de reglas.

7. Terceros

AdGuard LATAM no comparte datos con terceros, no integra redes publicitarias, ni utiliza servicios de analítica de comportamiento de usuarios.

La única conexión externa es la descarga de configuración desde GitHub (descrita en la sección 4), que no involucra datos del usuario.

8. Menores de edad

Esta extensión está dirigida a profesionales de marketing digital. No recopilamos datos de ninguna persona, incluyendo menores de edad.

9. Cambios a esta política

Si esta política cambia de forma relevante (por ejemplo, si se activa la telemetría), se notificará en las notas de actualización de la extensión en Chrome Web Store y se actualizará la fecha de vigencia en este documento.
